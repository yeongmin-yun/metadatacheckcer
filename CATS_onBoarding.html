2. Mouse API

2.1. dispatchMouseEvent

Mouse를 실제로 제어하지 않으면서 마우스가 동작하는 것처럼 에뮬레이션하여 테스트를 할 수 있는 메소드입니다. 테스트 대상의 컴포넌트에 마우스 클릭, 마우스 휠스크롤을 사용할 수 있습니다.

dispatchMouseEventEx는 dispatchMouseEvent 함수를 확장한 함수입니다.

Supported Environments

  • Desktop: Windows nexacro Browser, Chrome, IE11
  • Mobile: iOS nexacro Browser, Android nexacro Browser

Syntax

nexacro.test.Event.dispatchMouseEvent(targetElem, type, buttonName, clientX, clientY, [relatedElem])
nexacro.test.Event.dispatchMouseEventEx(targetElem(comp), type, buttonName, altKey, ctrlKey, shiftKey, clientX, clientY, [relatedElem])
nx_common_dispatchMouseEvent(targetElem(comp), type, buttonName, altKey, ctrlKey, shiftKey, clientX, clientY, [relatedElem])

nx_common_dispatchMouseEvent는 Desktop과 Mobile에서 공통으로 사용할 수 있는 래퍼 함수입니다.

Parameters

Parameter Type Description
targetElemObject마우스가 입력될 ControlElement 또는 Component Object.
typeString마우스 이벤트 유형: "lbuttondown", "lbuttonup", "rbuttondown", "rbuttonup", "mouseenter", "mousemove", "mouseleave", "mousewheel", "dblclick", "mousedown", "mouseup"
buttonNameString마우스 버튼: "lbutton", "rbutton", "mbutton". type이 'mousewheel'인 경우 빈 값("") 사용.
ctrlKeyBooleanControl 키 입력 유무 (true/false)
shiftKeyBooleanShift 키 입력 유무 (true/false)
altKeyBooleanAlt 키 입력 유무 (true/false)
clientXNumber이벤트의 X 좌표
clientYNumber이벤트의 Y 좌표
relatedElemObject이벤트와 연관된 ControlElement Object (예: mouseenter, mouseleave)

Return

Mouse event object를 반환하며, 해당 object의 내부 파라미터 값은 브라우저마다 다를 수 있습니다.

Remark

click(rbutton) 파라미터 사용 시 usercontextmenu 프로퍼티가 true이면 contextmenu가 열리므로, 테스트 자동화 샘플 작성 시 false로 설정해야 합니다.

Sample Call

// lbuttondown
var target = this.div00.form.edt00.getElement();
nexacro.test.Event.dispatchMouseEvent(target, 'lbuttondown', 'lbutton', 15, 20);
var comp = this.div00.form.edt00;
nexacro.test.Event.dispatchMouseEventEx(comp, 'lbuttondown', 'lbutton', false, false, false, 15, 20);

// rbuttondown
var target = this.div00.form.txt00.getElement();
nexacro.test.Event.dispatchMouseEvent(target, 'rbuttondown', 'rbutton', 15, 20);

// mouseenter
var target = this.div00.form.btn00.getElement();
var comp = this.div00.form.btn00;
var relatedTarget = this.div00.form.getElement();
nexacro.test.Event.dispatchMouseEvent(target, 'mouseenter', '', 10, 20, relatedTarget);

// dblclick
var target = this.div00.form.btn01.getElement();
setTimeout(function () {
    nexacro.test.Event.dispatchMouseEvent(target, 'dblclick', 'lbutton', 15, 20);
}, 50);

2.2. click

마우스 클릭 이벤트를 테스트하는 API입니다. 타이머 로직 처리 부분을 개발하기 쉽게 구현한 메소드입니다.

Supported Environments

  • Desktop: Windows nexacro Browser, Chrome, IE11
  • Mobile: iOS nexacro Browser, Android nexacro Browser

Syntax

nexacro.test.Event.click(targetElem, buttonName, clientX, clientY)
nexacro.test.Event.clickEx(targetElem(comp), buttonName, altKey, ctrlKey, shiftKey, clientX, clientY)
nexacro.test.Event.nx_common_click(targetElem(comp), buttonName, altKey, ctrlKey, shiftKey, clientX, clientY)

Parameters

Parameter Type Description
targetElemObject마우스가 입력될 ControlElement 또는 Component Object.
buttonNameString마우스 버튼: "lbutton"
altKeyBooleanAlt 키 입력 유무 (true/false)
ctrlKeyBooleanControl 키 입력 유무 (true/false)
shiftKeyBooleanShift 키 입력 유무 (true/false)
clientXNumber마우스가 입력될 X 좌표
clientYNumber마우스가 입력될 Y 좌표

Sample Call

// Click with lbutton
var target = this.div00.form.btn00.getElement();
setTimeout(function () {
    var rtn = nexacro.test.Event.click(target, 'lbutton', 10, 10);
});

var comp = this.div00.form.btn00;
setTimeout(function () {
    var rtn = nexacro.test.Event.clickEx(comp, 'lbutton', false, false, false, 10, 10);
});

Return

없음.

Remark

  • nexacro.test.Event.click은 element만 지정할 수 있으며, alt, ctrl, shift key를 제어할 수 없습니다.
  • nexacro.test.Event.clickEx은 component object를 지정할 수 있고, alt, ctrl, shift key를 지정할 수 있습니다.

3. Touch API

3.1. dispatchTouchEvent

Touch를 실제 제어하지 않으면서 터치를 에뮬레이션하는 메소드입니다. 테스트 대상의 터치 이벤트(탭, 더블탭, 스와이프)를 발생시킬 수 있습니다.

Supported Environments

  • Desktop: Windows nexacro Browser, Chrome, IE11
  • Mobile: iOSnexacro Browser, Android nexacro Browser

Syntax

nexacro.test.Event.dispatchTouchEvent(targetElem, type, identifier, clientX, clientY)

Parameters

Parameter Type Description
targetElemObject화면에서 터치할 ControlElement 또는 Component Object.
typeString터치 형태: "touchstart", "touchmove", "touchend"
identifierNumber이벤트를 일으키는 포인터를 위한 고유 식별자 (0, 1, 2, 3...).
clientXNumber화면에서 터치할 X 좌표.
clientYNumber화면에서 터치할 Y 좌표.

Return

Touch event object를 반환하며, 해당 object의 내부 파라미터 값은 브라우저마다 다를 수 있습니다.

Remark

없음.

Sample Call

// Tab
var target = this.div00.form.btn00.getElement();
nexacro.test.Event.dispatchTouchEvent(target, 'touchstart', 0);
nexacro.test.Event.dispatchTouchEvent(target, 'touchend', 0);

// Double Tab
var target = this.div00.form.btn02.getElement();
setTimeout(function () {
    nexacro.test.Event.dispatchTouchEvent(target, 'touchstart', 0);
    nexacro.test.Event.dispatchTouchEvent(target, 'touchend', 0);
    nexacro.test.Event.dispatchTouchEvent(target, 'touchstart', 0);
    nexacro.test.Event.dispatchTouchEvent(target, 'touchend', 0);
}, 100);

// Swipe
var target = this.div00.form.btn03;
var target_elem = target.getElement();
var offsetLeft = target.getOffsetLeft();
var offsetTop = target.getOffsetTop();
nexacro.test.Event.dispatchTouchEvent(target_elem, 'touchstart', 0, offsetLeft, offsetTop);
nexacro.test.Event.dispatchTouchEvent(target_elem, 'touchmove', 0, offsetLeft - 10, offsetTop);
nexacro.test.Event.dispatchTouchEvent(target_elem, 'touchend', 0, offsetLeft - 10, offsetTop);

4. Keyboard API

4.1. dispatchKeyBoardEvent

키보드를 실제로 제어하지 않으면서 키보드가 동작하는 것처럼 문자 키 입력을 에뮬레이션하는 메소드입니다.

Syntax

nexacro.test.Event.dispatchKeyBoardEvent(elem, name, key, [location, [ctrlkey, [shiftkey]]], [altkey])
nexacro.test.Event.dispatchKeyBoardEventEx(elem(comp), name, key, location, ctrlkey, shiftkey, altkey, clientX, clientY)

4.2. sendKeys

키보드를 실제로 제어하지 않으면서 키보드가 동작하는 것처럼 문자열 입력을 에뮬레이션하는 메소드입니다.

Syntax

nexacro.test.Event.sendKeys(element(comp), keys, [pos, [range]])

Parameters

Parameter Type Description
element(comp)Object키보드가 입력될 ControlElement 또는 Component Object.
keysArray입력할 문자열. 예: ['a'], ['ab\\n', 'd', nexacro.Event.KEY_LEFT]
posNumber입력할 문자열의 위치. "start" 또는 "end".
rangeArray입력할 문자열의 위치를 길이로 지정. 예: [0, 3]

Remark

일부 특수문자("#$%'&")의 입력은 불가능합니다.

4.3. key_downup

키 입력을 간소화한 API로, keydown과 keyup을 한 번에 수행합니다.

Syntax

nexacro.test.Event.key_downup(targetElem(comp), key, location, ctrlKey, shiftKey, altKey, clientX, clientY)

Parameters

Parameter Type Description
targetElem(comp)Object키보드가 입력될 ControlElement 또는 Component Object.
keyString입력할 키보드 버튼. 예: "Enter", "A", "a", "ArrowLeft" 등.
locationString키의 위치. 0: default, 1: LEFT, 2: RIGHT, 3: NUMPAD.
ctrlKeyBooleanControl 키 입력 유무.
shiftKeyBooleanShift 키 입력 유무.
altKeyBooleanAlt 키 입력 유무.

5. Style API

5.1. getComputedStyle

Component Property 속성 중 Style을 검증하는 API입니다. Computed style 값을 통해 이미지 유효성을 검증할 수 있습니다.

Syntax

nexacro.test.Style.getComputedStyle(obj, prop [, clientX, clientY])

Return

스타일 값은 객체이거나 문자열이며, 컴포넌트 종류에 따라 지원하지 않는 스타일도 존재합니다. 속성 값이 두 개 이상이면 객체 타입으로 반환됩니다. (예: border)

5.2. getComputedStyleString

Component Property 속성 중 Style을 검증하는 API로, 항상 문자열 값을 반환합니다.

Syntax

nexacro.test.Style.getComputedStyleString(obj, prop [, clientX, clientY])

Return

스타일 값은 항상 문자열이며, 컴포넌트 종류에 따라 지원하지 않는 스타일도 존재합니다. 포지션 별 동일 값이 있는 경우 CSS 룰에 따라 값이 축약될 수 있습니다.

6. Misc API

6.1. nx_key_copy

키보드 Ctrl+C 입력의 키보드 이벤트를 발생시킵니다.

nexacro.test.Event.nx_key_copy(comp)

6.2. nx_key_cut

키보드 Ctrl+X 입력의 키보드 이벤트를 발생시킵니다.

nexacro.test.Event.nx_key_cut(comp)

6.3. nx_key_all

키보드 Ctrl+A 입력의 키보드 이벤트를 발생시킵니다.

nexacro.test.Event.nx_key_all(comp)

6.4. nx_key_paste

키보드 Ctrl+V 입력의 키보드 이벤트를 발생시킵니다.

nexacro.test.Event.nx_key_paste(comp, paste_string)

6.5. nx_dragdrop

마우스 드래그 앤 드롭 기능을 스크립트로 테스트합니다.

nexacro.test.Event.nx_dragdrop(drag_elem, drop_elem, related_elem, drag_clientX, drag_clientY, drop_clientX, drop_clientY, related_clientX, related_clientY)

6.6. nx_getGridEditor

Grid Cell이 Edit 타입일 때 editObject를 구하는 API입니다.

nexacro.test.Event.nx_getGridEditor(comp)

6.7. nx_toColorname

color 값의 hexadecimal 값을 Named Color 값으로 변환하는 API입니다.

nexacro.test.Style.nx_toColorname(hexadecimal)

6.8. clearNXData

테스트 샘플 종료 후 다음 테스트로 넘어가기 전 초기화 작업을 수행하는 API입니다.

nexacro.test.Event.clearNXData()

7. API 사용시 주의사항

7.1. 컴포넌트에 포함된 scrollbar 사용

Scrollbar를 가진 컴포넌트를 클릭하는 API 사용 시, component.hscrollbar 또는 component.vscrollbar object를 매개변수로 사용해야 정상 동작합니다.

7.2. popup창 접근

Popup창을 띄워 API 동작을 수행시키는 경우, popup창의 객체(component.popupwindow)를 넘기거나 element를 넘기는 방식을 추천합니다. Menu 사용 시 popupmenu는 1단계만 지원됩니다.
